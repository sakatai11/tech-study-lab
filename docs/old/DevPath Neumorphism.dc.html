XSSの基礎とエスケープ設計

Security / XSS約18分4択3問

XSSは「入力値が危険」なのではなく、値がHTMLやJavaScriptとして解釈される文脈に混入することで成立します。実装では保存前の一律除去より、出力時の文脈別エスケープを軸に考えます。
